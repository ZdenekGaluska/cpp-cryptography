# C++ Cryptography — OpenSSL EVP API
 
Cryptography implementations using the OpenSSL EVP API.
 
- `openssl_file_encryption.cpp` — symmetric file encryption/decryption (AES-ECB, AES-CBC)
- `openssl_hash_prefix.cpp` — brute-force search for a message with a given hash bit prefix
- `openssl_hybrid_encryption.cpp` — hybrid encryption using RSA + AES (seal/open)
Coursework from FIT CTU Prague — BI-KAB (Cryptography)
